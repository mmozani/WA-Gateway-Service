# 🚀 WA-Gateway-Service (Professional Multi-Session WhatsApp API)

[![Node.js Version](https://img.shields.io/badge/node-%3E%3D%2018.0.0-brightgreen)](https://nodejs.org/)
[![License](https://img.shields.io/badge/license-MIT-blue)](LICENSE)

یک راهکار سازمانی و میکروسرویس قدرتمند برای تبدیل واتس‌اپ به یک **دروازه ارسال پیامک (SMS Gateway)**. این سرویس به کسب‌وکارهای ایرانی اجازه می‌دهد بدون هزینه‌های دلاری و سنگینِ پیامک‌های بین‌المللی، با مشتریان خود در سراسر جهان در ارتباط باشند.

---

## 🌟 چرا این سرویس برای کسب‌وکارهای ایرانی حیاتی است؟

ارسال کد تایید (OTP) به شماره‌های خارج از کشور از داخل ایران با چالش‌های بزرگی روبروست:
* **هزینه‌های گزاف ارزی:** قیمت هر پیامک بین‌المللی با نوسان دلار تغییر کرده و بسیار گران است. این سرویس هزینه را به **صفر** می‌رساند.
* **تحریم سرویس‌دهنده‌ها:** شرکت‌هایی مثل Twilio به ایران خدمات نمی‌دهند. با این پروژه، شما مالک زیرساخت خود هستید.
* **نرخ تحویل (Delivery Rate):** بسیاری از پیامک‌های بین‌المللی توسط اپراتورهای خارجی بلاک می‌شوند، اما پیام واتس‌اپ ۱۰۰٪ به دست کاربر می‌رسد.

---

## 📱 پیش‌نیازها: اقدامات ضروری روی گوشی موبایل

قبل از اجرای سرویس، حتماً تنظیمات زیر را روی گوشی فرستنده انجام دهید تا پایداری سشن تضمین شود:

1.  **Linked Devices:** وارد واتس‌اپ گوشی شوید، به بخش `Linked Devices` بروید و مطمئن شوید نسخه واتس‌اپ شما به‌روز است.
2.  **بهینه‌سازی باتری (Battery Optimization):** در تنظیمات گوشی، برنامه واتس‌اپ را از لیست `Battery Optimization` خارج کنید (گزینه Don't Optimize را انتخاب کنید) تا سیستم عامل ارتباط واتس‌اپ را در پس‌زمینه قطع نکند.
3.  **اتصال اینترنت:** اگرچه این سرویس از نسخه Multi-Device استفاده می‌کند، اما گوشی فرستنده باید حداقل هر ۱۴ روز یک بار به اینترنت متصل شود تا سشن باطل نشود.
4.  **استفاده از نسخه اصلی:** حتماً از واتس‌اپ نسخه اصلی یا Business استفاده کنید (از نسخه‌های غیررسمی مثل جی‌بی واتس‌اپ استفاده نکنید).

## ✨ امکانات پیشرفته (Features)

* **Anti-Ban System:** پیاده‌سازی مکانیزم تأخیر تصادفی (۲ تا ۵ ثانیه) بین ارسال‌ها برای شبیه‌سازی رفتار انسانی و جلوگیری از مسدود شدن خط.
* **Multi-Session Management:** مدیریت نامحدود شماره موبایل به صورت ایزوله روی یک پورت.
* **Webhooks:** گزارش‌دهی لحظه‌ای رویدادها (وصل شدن، قطع شدن، تولید QR) به پنل مدیریت شما (مانند Laravel).
* **Health Monitoring:** مشاهده آپ‌تایم، وضعیت حافظه RAM و سلامت سرویس از طریق API.
* **Auto-Recovery:** تلاش هوشمند برای راه‌اندازی مجدد سشن‌ها در صورت بروز خطای شبکه.
* **Security:** لایه امنیتی API Key برای جلوگیری از دسترسی غیرمجاز.

## 🛠 نصب و راه‌اندازی (لینوکس / اوبونتو)

### روش اول: نصب سریع با اسکریپت اختصاصی
ما یک اسکریپت آماده کرده‌ایم که تمام پیش‌نیازهای لینوکس و کتابخانه‌های گرافیکی مورد نیاز برای شبیه‌ساز مرورگر را به صورت خودکار نصب می‌کند:

```bash
# ۱. دریافت پروژه
git clone [https://github.com/mmozani/WA-Gateway-Service.git](https://github.com/mmozani/WA-Gateway-Service.git)
cd WA-Gateway-Service

# ۲. اجرای اسکریپت نصب
chmod +x wa-gateway-service.sh
./wa-gateway-service.sh
```

### روش دوم: نصب دستی
اگر می‌خواهید مراحل را دستی بروید:
۱. `npm install` را بزنید.
۲. فایل `.env.example` را به `.env` تغییر نام داده و مقادیر را پر کنید.

## 🚀 اجرا و مدیریت (Production)

برای اجرای پایدار در سرور لینوکس، از **PM2** استفاده می‌کنیم تا سرویس در صورت کرش یا ریبوت سرور، فوراً بالا بیاید:

```bash
# شروع سرویس با تنظیمات بهینه رم (500MB)
pm2 start ecosystem.config.js

# ذخیره وضعیت برای بالا آمدن خودکار پس از ریبوت سرور
pm2 save
pm2 startup
```

## 📡 مستندات API (Routes)

**هدر الزامی:** `X-API-KEY: your_secret_key`

### ۱. ارسال پیام (OTP)
* **Route:** `POST /send-otp`
* **Body:**
```json
{
  "phone": "989120000000",
  "message": "کد تایید شما: 1234",
  "session_id": "line-1"
}
```

### ۲. وضعیت لحظه‌ای خطوط
* **Route:** `GET /status`
* **توضیح:** نمایش لیست شماره‌های متصل، وضعیت اتصال و زمان آماده‌باش.

### ۳. مانیتورینگ سلامت
* **Route:** `GET /health`
* **توضیح:** نمایش وضعیت رم، آپ‌تایم سرور و زمان فعلی سیستم.

### ۴. حذف و ریست سشن
* **Route:** `DELETE /session/:id`
* **توضیح:** قطع اتصال کامل، پاکسازی کش و آماده‌سازی برای اسکن مجدد کد QR.
